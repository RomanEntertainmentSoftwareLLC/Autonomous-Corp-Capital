## APPROVAL CHECKS

- [ ] Run ID assigned and unique
- [ ] Date timestamp valid (ISO8601)
- [ ] Task was runtime work (not chatter)
- [ ] All actions have proof references
- [ ] Proof references are verifiable (files exist, logs show results)
- [ ] XP math is correct per RPG_RULES.md
- [ ] Rounding rule applied (nearest integer, 0.5 rounds up)
- [ ] Only 3 active stats used (ACC, REL, JUD)
- [ ] Speed (SPD) not scored (dormant in v1)
- [ ] Penalties documented if applicable
- [ ] Reviewer name recorded
